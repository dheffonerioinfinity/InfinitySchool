POLÍTICA DE PRIVACIDADE

Aplicativo: InfinitySchool

Controlador de Dados: Iniciativa Independente / Colégio Estadual Dr. Onério Pereira Vieira

Vigência: A partir de 07 de Fevereiro de 2026

1. INTRODUÇÃO E ESCOPO

Esta Política de Privacidade descreve como o aplicativo InfinitySchool coleta, armazena e processa dados. O aplicativo é uma ferramenta tecnológica desenvolvida para fins informativos e pedagógicos no âmbito do Colégio Estadual Dr. Onério Pereira Vieira (Quirinópolis-GO).

Reiteramos que esta solução é independente e, embora autorizada pela gestão escolar, não possui vínculo oficial com o Governo do Estado de Goiás ou suas secretarias, sendo a privacidade dos dados gerida pela equipe de desenvolvimento local.

2. COLETA DE DADOS DOS ALUNOS (USUÁRIOS FINAIS)

O InfinitySchool preza pelo anonimato e privacidade dos alunos. Para a utilização do aplicativo por parte dos estudantes:

  • Não há cadastro: Não solicitamos nome completo, CPF, e-mail, número de matrícula ou documentos pessoais.
  • Dados de Navegação: O aplicativo coleta apenas a seleção de preferências (Série, Turma e Turno) para filtrar o conteúdo exibido.
  • Identificadores de Dispositivo (Tokens): Para o funcionamento do sistema de notificações (avisos de provas, comunicados, etc.), o aplicativo utiliza serviços de terceiros (OneSignal e Firebase Cloud Messaging) que geram um token anônimo vinculado ao dispositivo, e não à pessoa do aluno.

3. DADOS DE PROFESSORES E GESTÃO DE ACESSO

O acesso administrativo para postagens é restrito e não permite auto-cadastro (sign-up) via aplicativo.

  • Natureza dos Dados: Os dados presentes no banco de dados referem-se estritamente à vida funcional pública do docente na unidade escolar: Primeiro Nome, Disciplinas Ministradas e Turmas.
  • Credenciais (Senhas/Tokens): As senhas de acesso são geradas aleatoriamente pela administração do sistema e entregues pessoalmente ao professor. Ao aceitar a credencial e realizar o login, o professor consente com o processamento desses dados mínimos para a execução das funcionalidades do app.

4. ARMAZENAMENTO E INFRAESTRUTURA (TERCEIROS)

O InfinitySchool não mantém servidores físicos próprios. Para garantir segurança, estabilidade e disponibilidade, utilizamos infraestrutura de nuvem de renome mundial.

Os dados (conteúdo das postagens, cardápios e referências de turmas) são armazenados e processados através dos serviços:

  • Google Firebase (Firestore & Realtime Database): Responsável pelo banco de dados em nuvem. Ao usar o app, seus dados técnicos trafegam pelos servidores seguros do Google.
  • OneSignal & Firebase Cloud Messaging: Responsáveis pelo envio e gestão de notificações push.

5. PERMISSÕES DO DISPOSITIVO

Para o pleno funcionamento, o aplicativo pode solicitar permissões específicas no sistema Android:

  • Permissão de Notificações [ANDROID 13+]: Para o app, através de sistemas de terceiros já abordados neste documento, enviar notificações push (push notifications) conforme atualizações do banco de dados ou informações locais.
  • Acesso à Internet: Indispensável para baixar atualizações de conteúdo.
  • Armazenamento/Cache: O aplicativo pode armazenar dados temporários (cache) no dispositivo do usuário para acelerar o carregamento e economizar dados móveis. O usuário tem total liberdade para limpar esses dados através das configurações do seu sistema operacional ou na seção "Configurações" do aplicativo.

6. SEGURANÇA DA INFORMAÇÃO

Adotamos medidas técnicas adequadas para proteger os dados, incluindo o uso de conexões criptografadas (HTTPS/SSL) entre o aplicativo e os servidores do Google Firebase. No entanto, ressalta-se que nenhum sistema é 100% inviolável, e o usuário compreende que o desenvolvedor não se responsabiliza por interceptações ilegais ou violações decorrentes de falhas de segurança no dispositivo do próprio usuário (ex: vírus, root, redes não seguras, etc.).

7. DIREITOS E EXCLUSÃO DE DADOS

Em conformidade com a LGPD:

  • Alunos: Como não há coleta de dados pessoais identificáveis, não há dados a serem excluídos. Basta desinstalar o aplicativo para cessar qualquer vínculo técnico.
  • Professores: O docente pode, a qualquer momento, solicitar o desvinculamento de sua conta e a revogação de seu token de acesso, bastando comunicar diretamente a administração do projeto na unidade escolar.

Ao instalar e utilizar o InfinitySchool, você declara estar ciente e de acordo com esta Política de Privacidade.
